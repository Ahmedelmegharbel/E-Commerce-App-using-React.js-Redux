# E-Commerce-App-using-React.js-Redux

E-Commerce application using React and Redux to display a list of products with an "Add to Cart" button for each product, display the items added to the cart, and allow users to remove items from the cart, increase and decrease items quantity in the cart, clear all cart by one button and display a loyalty program supercoins for each cart. All this information is going to be globally available throughout the application using Redux Toolkit.
